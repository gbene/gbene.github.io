• Interface overview

The main window of PZero is composed by three main interfaces:

  1. The menu.
  2. The toolbar.
  3. The project explorer.

The menu and toolbar

The menu at the top of the window (A) groups all of the available PZero functions and viewers. The main categories are:

  1. File: Input and output functions
  2. Edit: Object managment functions (properties, textures etcetc)
  3. Interpolation: Interpolation and object manipulation functions
  4. Window: Different viewers (3D, map view, section view etcetc)

The same subdivisions and functions are present in the toolbar (B).
