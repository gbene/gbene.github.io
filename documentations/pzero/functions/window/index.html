4. Window

The Window category encapsulates all of the different functions that can be used in the viewers. There are three main types of viewers:

  1. 3D viewer: View object in 3D space
  2. Map view: View objects from a planar prespective perpendicular to the XY plane
  3. Cross section view: View the different crossections (if present)
