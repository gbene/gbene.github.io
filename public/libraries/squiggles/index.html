Squiggles

Posted on Apr 19, 2026 First author

Squiggles.jl is a Julia package to accelerate 1D cross correlations of seismic signals using GPUs! We use KernelAbstractions.jl to run on any GPU supported by the JuliaGPU ecosystem.
